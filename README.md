Aquafalx\
\
This is a project to learn go, git, network generation, and cyber offense and defense.\
\
The goal is to make a game where all of the units are computers with randomized vulnerabilies that can be hacked.\
\
Table of Contents:\
    1. Installation\
    2. Usage\

Installation:\
    cd [PATH to Aquafalx]\
    go install .\
\
Usage:\
    cd src\
    cd app\
    go run Init.go\
